# NetPulse

**Автоматизированная система для снижения социальной тревожности и организации совместной деятельности.**

Учебный проект по дисциплине «Технологии программирования». Клиентское iOS-приложение взаимодействует с сервером по WebSocket (реальное время) и REST API; архитектура — Clean Architecture + MVVM.

---

## Назначение

- Снижение социальной тревожности за счёт честных статусов («Онлайн» / «Не в сети» и др.).
- Организация синхронизированных по времени парных и групповых активностей: работа, учёба, медитация.

## Требования

- **iOS** 17.0+
- **Xcode** с поддержкой Swift 5
- **Язык интерфейса:** русский

## Стек

| Клиент | Сервер (по ТЗ) |
|--------|-----------------|
| Swift, SwiftUI, Combine | Go, WebSocket |
| Clean Architecture, MVVM | In-memory → MySQL |

## Запуск

1. Клонировать репозиторий.
2. Открыть `NetPulse.xcodeproj` в Xcode.
3. Выбрать симулятор или устройство, запустить (⌘R).

## Функционал (по ТЗ)

- [x] Регистрация по уникальному имени (username/email).
- [x] Установка предопределённых статусов (Онлайн, Оффлайн, Работаю, Учусь).
- [x] Список друзей с актуальными статусами.
- [x] Поиск и добавление пользователей в друзья.
- [x] Отправка/приём приглашений на сессию (только пользователям «Онлайн»).
- [x] Уведомления о входящих приглашениях (принять/отклонить).
- [x] Синхронизированный таймер для участников сессии (запуск по принятию приглашения, вкладка «Сессия»).
- [ ] Текстовый чат в рамках сессии.
- [ ] Достижения за завершённые сессии.
